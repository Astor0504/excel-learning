WPS JS 宏——把你會的 VBA 平移過去

本課進度 0 / 4
TL;DR — 30 秒看懂

WPS 免費版沒有 VBA，但內建「JS 宏」：語法換成 JavaScript，物件模型跟 VBA 幾乎同一套。學完 P5-01～03 的你只要換三個習慣就能上手——連 macOS 免費版都能用。

這是一堂番外課，寫給平常用 WPS 的你。前面三課的 VBA 觀念——流程拆解、錯誤處理、陣列加速——一個都不會浪費，這課只是幫它們換一個語法的殼。

為什麼 WPS 用戶需要這一課

WPS 免費版不含 VBA 模組，P5-01～03 的巨集在 WPS 裡按了沒反應是正常的。WPS 官方的自動化路線是內建的 JS 宏（JSA）：用 JavaScript 語法，操作跟 VBA 同一套物件模型（Application、Worksheets、Range）。macOS 國際版免費 WPS（v12.1 實測）就內建，不用另外裝任何元件。

打開 JS 宏編輯器

在 WPS 表格裡找「開發工具」分頁 →「JS 宏」（不同版本入口名稱略有差異，可能在「工具」選單底下；找不到就用選單搜尋輸入「宏」）。打開後新增一個模組，就可以寫第一個宏了。

第一個 JS 宏

function 第一個宏() {
    Range("A1").Value2 = "Hello WPS";
    alert("寫入完成！");
}

跟 VBA 的三個第一眼差異：不用 Dim 宣告型別、alert() 取代 MsgBox、註解用 //。游標放在函式裡按執行鍵就會跑。

範例：迴圈與最後一列

VBA 課教過的 End(xlUp) 找最後一列，在 JS 宏裡長得幾乎一樣：

function 標記缺交() {
    let sh = Worksheets.Item("名單");
    let lastRow = sh.Cells.Item(sh.Rows.Count, "A").End(xlUp).Row;
    for (let r = 2; r <= lastRow; r++) {
        if (!sh.Cells.Item(r, 2).Value2) {        // B 欄空白視為未填
            sh.Cells.Item(r, 3).Value2 = "缺交";
        }
    }
}

範例：錯誤處理與設定復原

P5-02 的「出錯也要把設定復原」原則，在 JS 宏用 try / catch / finally 實現——finally 不管成功失敗都會執行，正是放復原程式碼的位置：

function 安全執行() {
    Application.ScreenUpdating = false;
    Application.DisplayAlerts = false;
    try {
        Worksheets.Item("月報").Range("A1").Value2 = "更新於 " + new Date().toLocaleString();
    } catch (err) {
        alert("流程失敗：" + err.message);
    } finally {
        Application.ScreenUpdating = true;
        Application.DisplayAlerts = true;
    }
}

範例：陣列一次讀寫

大量資料逐格讀寫一樣會卡。JS 宏的解法跟 VBA 課同一招：Value2 一次讀進二維陣列、處理完用 Resize 一次寫回：

function 陣列加速() {
    let sh = Worksheets.Item("資料");
    let lastRow = sh.Cells.Item(sh.Rows.Count, "A").End(xlUp).Row;
    let data = sh.Range("A2:B" + lastRow).Value2;      // 一次讀進二維陣列
    let result = [];
    for (let i = 0; i < data.length; i++) {
        result.push([data[i][0] + "-" + data[i][1]]);
    }
    sh.Range("C2").Resize(result.length, 1).Value2 = result;  // 一次寫回
}

實戰：跨工作表彙總

把整本活頁簿裡名稱符合規則的工作表掃一遍、彙總到同一張表——月報、分店報表最常見的需求，也是前面所有招式的合體：

function 彙總各分店() {
    Application.ScreenUpdating = false;
    try {
        let target = Worksheets.Item("彙總");
        let result = [];
        for (let s = 1; s <= Worksheets.Count; s++) {
            let sh = Worksheets.Item(s);
            let name = sh.Name.trim();
            if (!name.startsWith("分店-")) continue;   // 只收分店工作表
            let lastRow = sh.Cells.Item(sh.Rows.Count, "B").End(xlUp).Row;
            if (lastRow < 2) continue;
            let data = sh.Range("A2:B" + lastRow).Value2;
            for (let i = 0; i < data.length; i++) {
                if (data[i][0]) result.push([name, data[i][0], data[i][1]]);
            }
        }
        if (result.length > 0) {
            target.Range("A2").Resize(result.length, 3).Value2 = result;
        } else {
            alert("找不到任何「分店-」開頭的工作表資料");
        }
    } catch (err) {
        alert("彙總失敗：" + err.message);
    } finally {
        Application.ScreenUpdating = true;
    }
}
