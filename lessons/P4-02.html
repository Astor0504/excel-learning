進階函數：解決傳統函數做不到的場景

這些函數平常用不到，但碰到時你會慶幸自己學過。

重點函數

=LET(x, A2*0.05, y, B2+x, y*1.1)
# 命名變數，避免重複計算同一個東西

=INDIRECT("工作表" & A2 & "!B5")
# 用文字組出儲存格參照（動態跨表查詢）

=OFFSET(A1, 5, 2, 1, 1)
# 從 A1 偏移 5 列 2 欄 → C6

=AGGREGATE(9, 7, A:A)
# 強化版 SUM，可忽略隱藏列、錯誤值

=SWITCH(A2, "A", 100, "B", 80, "C", 60, 0)
# 比巢狀 IF 好讀的多分支

=LAMBDA(x, y, x^2 + y^2)(3, 4)
# 自訂函數（M365 only），可放在「名稱管理員」變成可重用函式

使用建議

LET 是現在最被推薦的「公式重構」工具——把 =A1*0.05 + (A1*0.05)*1.1 改成 =LET(t, A1*0.05, t + t*1.1) 容易讀又只算一次。
