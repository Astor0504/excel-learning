Power Query：Excel 內建的 ETL 神器

再也不用每月手動清資料！Power Query 把整理流程錄起來，下個月一鍵重跑。

什麼是 ETL

Extract（取資料）→ Transform（清資料）→ Load（載入 Excel）。Power Query 把這三步全部視覺化，不用寫程式碼。

常見操作

資料 → 取得資料 → 從檔案/資料庫/網頁。匯入後進入 Power Query 編輯器：可以拆欄、合併欄、移除列、轉置、樞紐/反樞紐、合併查詢（類似 JOIN）、附加查詢（類似 UNION）。

M 語言

let
    來源 = Excel.CurrentWorkbook(){[Name="訂單"]}[Content],
    篩選 = Table.SelectRows(來源, each [金額] > 1000),
    排序 = Table.Sort(篩選, {{"金額", Order.Descending}})
in
    排序

為什麼要用

傳統複製貼上的工作流程：每個月花 2 小時。用 Power Query 第一次設定 30 分鐘，之後每月「重新整理」按一下，10 秒搞定。
