VBA 基礎：用程式碼自動化重複工作

VBA 是 Excel 的內建程式語言。會寫一點 VBA，等於有個 24 小時不抱怨的助理。

⚠️ 注意Mac 沒有 Alt+F11，要用「工具 → 巨集 → Visual Basic 編輯器」進入。Mac 版不支援 UserForm（自訂表單），其他基礎功能都能用。

第一個巨集

Sub HelloWorld()
    MsgBox "哈囉，VBA！"
End Sub

Sub 寫入儲存格()
    Range("A1").Value = "我是 VBA 寫的"
    Range("A2").Value = 100
    Range("A3").Formula = "=A2*1.05"
End Sub

變數與迴圈

Sub 加總範例()
    Dim i As Integer
    Dim total As Double
    total = 0
    
    For i = 2 To 100
        If Cells(i, 2).Value > 0 Then
            total = total + Cells(i, 2).Value
        End If
    Next i
    
    MsgBox "加總：" & total
End Sub

執行巨集

按 F5 執行；或回 Excel → 工具 → 巨集 → 選一個來執行；或建立按鈕綁定巨集。檔案要存成 .xlsm（啟用巨集的活頁簿）。
