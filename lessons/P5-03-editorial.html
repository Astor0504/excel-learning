自動化流程總覽

範例：月報交付自動化

工具分工提醒這個範例的重點是 VBA 負責的部分：開啟檔案 → 觸發更新 → 控制輸出。清資料（刪空白、重複值、格式整形）應該先在 Power Query 裡建好可重跑步驟，再讓 VBA 呼叫 RefreshAll。如果你把清資料邏輯寫進 VBA，每次流程變動就要改巨集，不如讓 Power Query 負責「清」，VBA 負責「跑」。
Sub 月報自動化()
    Application.ScreenUpdating = False  ' 關閉螢幕更新加速

    ' 1. 重新整理所有 Power Query 查詢（清資料邏輯留在 PQ）
    ActiveWorkbook.RefreshAll

    ' 2. 等待重整完成（若有非同步查詢）
    Application.CalculateUntilAsyncQueriesDone

    ' 3. 切到月報工作表，輸出 PDF
    Worksheets("月報").Activate
    ActiveSheet.ExportAsFixedFormat Type:=xlTypePDF, _
        Filename:="~/Desktop/月報_" & Format(Date, "yyyymm") & ".pdf"

    Application.ScreenUpdating = True
    MsgBox "月報已更新並輸出！"
End Sub

這個結構讓 VBA 只負責觸發與交付，清資料的責任留在 Power Query——改清資料規則時不用動巨集，只要改 Query 步驟即可。

加速三件事

① Application.ScreenUpdating = False（關螢幕更新）② Application.Calculation = xlCalculationManual（暫停自動計算）③ 用陣列代替逐格操作。

按鈕綁巨集

開發人員 → 插入 → 按鈕 → 拖出來 → 指定巨集。使用者就能一鍵執行。
