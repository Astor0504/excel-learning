VBA 進階：陣列、字典、錯誤處理、事件

進階 VBA 的精髓在「速度」——把 100 個操作壓成 1 個記憶體運算。

陣列加速

' 慢：直接讀寫儲存格（每次都跨應用程式）
For i = 2 To 10000
    Cells(i, 2).Value = Cells(i, 1).Value * 2
Next i

' 快：先讀進陣列，運算完一次寫回（快 100 倍）
Dim arr As Variant
arr = Range("A2:A10000").Value
For i = 1 To UBound(arr)
    arr(i, 1) = arr(i, 1) * 2
Next i
Range("B2:B10000").Value = arr

Dictionary（去重 / 計數神器）

Dim dict As Object
Set dict = CreateObject("Scripting.Dictionary")

For i = 2 To 1000
    Dim key As String
    key = Cells(i, 1).Value
    If dict.Exists(key) Then
        dict(key) = dict(key) + 1
    Else
        dict.Add key, 1
    End If
Next i

錯誤處理

Sub Safe()
    On Error GoTo ErrHandler
    ' 你的程式碼
    Exit Sub
ErrHandler:
    MsgBox "錯誤：" & Err.Description
End Sub

事件驅動

工作表可以掛事件：Worksheet_Change（資料變更時觸發）、Worksheet_SelectionChange（選取變更時）。可以做「自動填入時間戳記」之類的功能。
