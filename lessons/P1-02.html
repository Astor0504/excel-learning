本課進度 0 / 3
TL;DR — 30 秒看懂

7 個最常用的統計函數，幾乎所有報表都從這裡開始

這是 Excel 的「九九乘法表」——背熟這幾個，才有資格往後學進階函數。

基礎函數速覽

函數用途範例
SUM加總=SUM(A2:A100)
AVERAGE平均=AVERAGE(B2:B100)
COUNT計算數字個數=COUNT(C2:C100)
COUNTA計算非空個數（含文字）=COUNTA(D2:D100)
MAX / MIN最大 / 最小=MAX(E2:E100)
MEDIAN中位數=MEDIAN(F2:F100)
LARGE / SMALL第 N 大 / 小=LARGE(G2:G100, 3) → 第 3 大

實戰範例

# 業績統計（A欄=業務員，B欄=金額）
=SUM(B:B)              # 全部業績
=AVERAGE(B2:B100)      # 平均單筆
=COUNT(B:B)            # 有幾筆業績
=COUNTA(A:A) - 1       # 業務員人數（扣標題）
=LARGE(B:B, 1)         # 最高一筆
=LARGE(B:B, 3)         # 第 3 高（找前三名）
⚠️ 注意COUNT 只算「數字」，文字會被忽略！要算筆數請用 COUNTA。AVERAGE 會自動忽略空白和文字，但會把 0 算進去。

⌨️ 本課常用快捷鍵

WPS Office 本課快捷鍵差異
  • 自動加總：Excel 用 ⌘ ⇧ T，WPS 無對應快捷鍵（需手動輸入 =SUM）
  • 填滿向下：Excel 用 ⌃ D，WPS 用 ⌘ D
  • 多格同時輸入：Excel 用 ⌘ ↩，WPS 用 Ctrl + Enter
