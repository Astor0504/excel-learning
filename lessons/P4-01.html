陣列與動態函式

本課進度 0 / 4
TL;DR — 30 秒看懂

動態陣列：一個公式吐出整片結果，搭配 # 溢出範圍建立會自動更新的清單

動態陣列不是多背幾個新函數，而是換一種做清單的方式：一個公式放在左上角，結果自動向外展開；來源資料變動，整片結果一起更新。

先掌握 3 個觀念

動態陣列的心智模型
觀念你要記住常見用途
Spill 溢出公式只輸入一次，結果會自動展開到多格動態名單、篩選結果、排行表
D2## 代表從 D2 開始的整個溢出範圍把動態結果餵給圖表、驗證清單或其他公式
條件陣列FILTER 的條件其實是一串 TRUE/FALSE多條件篩選、清單自動更新

核心語法

# 篩選符合條件的整列；第 3 參數是找不到時的顯示文字
=FILTER(A2:E100, B2:B100="業務", "查無資料")

# 多條件 AND：兩個條件都要成立，用 * 相乘
=FILTER(A2:E100, (B2:B100="業務")*(E2:E100>=90), "查無資料")

# 先篩選，再依第 3 欄由大到小排序
=SORT(FILTER(A2:E100, B2:B100="業務"), 3, -1)

# 產生不重複清單，再排序
=SORT(UNIQUE(B2:B100))

# 引用從 D2 開始的整個動態結果
=COUNTA(D2#)

實戰流程：從原始表到自動清單

職場最常見的做法不是把每個函數分開炫技，而是用一條公式把「篩選 → 排序 → 輸出」串起來。例如主管要看業務部績效 90 分以上的人，你可以用 FILTER 先挑列，再用 SORT 依績效排序。

=SORT(
  FILTER(A2:E100, (B2:B100="業務")*(E2:E100>=90), "查無資料"),
  5,
  -1
)

最常見的 3 個錯誤

#SPILL!：結果要展開，但旁邊被擋住

動態陣列會往下或往右溢出。若輸出範圍裡有任何內容，Excel 會顯示 #SPILL!。先清空輸出區，再讓公式自己展開。

#CALC!：FILTER 找不到資料

FILTER 沒有符合結果、又沒寫第 3 參數時，會顯示 #CALC!。交付報表請加上「查無資料」這類可讀訊息。

#NAME?：版本不支援

FILTER、SORT、UNIQUE 主要支援 Microsoft 365 / Excel 2021 以後版本。給舊版使用者前，請先確認環境或準備 Power Query、樞紐分析等替代方案。

SUMPRODUCT 還需要學嗎？

需要。SUMPRODUCT 是舊版 Excel 也能使用的「條件陣列」思路，適合做條件加總或加權平均；但如果你要輸出整片明細清單，動態陣列會更直覺。

# 舊版也可用：電子部且金額 > 1000 的加總
=SUMPRODUCT((A:A="電子")*(B:B>1000)*C:C)
⚠️ 交付提醒原始資料建議放在 Table，動態陣列公式寫在表格外；不要把會溢出的公式塞進 Table 內部，否則很容易和表格自動擴張互相打架。

⌨️ 本課常用快捷鍵

WPS Office 動態陣列相容性
  • WPS 從 2022 版開始支援 FILTER、SORT、UNIQUE 等動態陣列函式
  • LAMBDA 函式在 WPS 中尚未支援
  • 動態陣列的「溢出」行為與 Excel 365 相同
