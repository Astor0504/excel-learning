動態陣列函數：M365 的革命，一個公式回傳整個範圍

這是 Excel 近 30 年最大的功能升級。會了之後再也回不去傳統公式了。

6 個必學動態函數

=FILTER(A2:C100, B2:B100="電子")
# → 篩選出電子部所有列（不用樞紐）

=SORT(A2:C100, 3, -1)
# → 依第 3 欄降序排序

=UNIQUE(A2:A100)
# → 取出不重複值

=SEQUENCE(10)
# → 生成 1~10 的序號

=SEQUENCE(12, 1, 1, 1)
# → 12 個月

=SORT(UNIQUE(FILTER(A2:A100, B2:B100>1000)))
# → 業績破千的人去重後排序，一行搞定！

溢出範圍 #

動態函數的結果會「溢出」到下方儲存格。寫 D2# 表示「從 D2 開始的整個溢出範圍」。

SUMPRODUCT（老牌動態）

# 沒有動態陣列也能玩
=SUMPRODUCT((A:A="電子")*(B:B>1000)*C:C)
# → 電子部且金額>1000 的加總
⚠️ 注意FILTER/SORT/UNIQUE 只在 Microsoft 365 / Excel 2021 以後才有。給舊版用會 #NAME?。
