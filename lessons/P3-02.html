文字與日期函式

本課進度 0 / 4
TL;DR — 30 秒看懂

文字 + 日期函數：清資料最常用的工具組

從別處貼過來的資料總是亂七八糟？這課的函數就是專門用來「整理髒資料」的。

文字函數

=TRIM(A2)              # 去除前後空白
=MID(A2, 3, 4)         # 從第 3 個字取 4 個
=LEFT(A2, 2)           # 取最左 2 個字
=RIGHT(A2, 2)          # 取最右 2 個字
=FIND("@", A2)         # 找符號位置
=SUBSTITUTE(A2, "-", "")  # 取代
=TEXT(A2, "yyyy-mm-dd")   # 數字/日期 → 格式化字串
=A2 & "-" & B2         # 串接（也可用 &）

日期函數

=TODAY()                       # 今天
=NOW()                         # 現在（含時間）
=DATEDIF(A2, TODAY(), "Y")    # 年資（年）
=DATEDIF(A2, TODAY(), "M")    # 月差
=DATEDIF(A2, TODAY(), "D")    # 天數
=WORKDAY(TODAY(), 5)          # 5 個工作天後
=EOMONTH(TODAY(), 0)          # 本月最後一天
=EOMONTH(TODAY(), -1)+1       # 本月第一天
⚠️ 注意DATEDIF 是隱藏函數，輸入時不會自動提示，但能用。MID 從 1 開始算，不是 0。
