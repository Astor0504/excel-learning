條件式格式化

本課進度 0 / 4
TL;DR — 30 秒看懂

條件式格式化：讓重要的資料自己跳出來

報表要給人看，不上色等於沒做。條件式格式可以自動把高/低值上色，看一眼就知道重點。

3 種視覺化

類型適合場景
資料橫條快速比較大小（像橫條圖）
色階熱力圖效果（紅高綠低）
圖示集加箭頭/燈號/星星

自訂公式規則（最強）

# 整列上色：當 D 欄是「逾期」就把整列變紅
# 選整列 → 條件式格式化 → 新增規則 → 使用公式
=$D2="逾期"
# ⚠️ $D2 加 $ 是鎖欄不鎖列，這樣每一列都會用自己的 D 欄判斷

# 高亮週末
=WEEKDAY($A2, 2)>=6

# 標記高於平均
=B2>AVERAGE($B$2:$B$100)
⚠️ 注意公式裡 $ 用錯地方是最常犯的錯。記住：要鎖「欄」就 $X，要鎖「列」就 X$。整列上色一定要鎖欄不鎖列。
