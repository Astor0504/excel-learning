查找比對

本課進度 0 / 4
TL;DR — 30 秒看懂

XLOOKUP / VLOOKUP / INDEX+MATCH：跨表查資料的命脈函數

查找比對不是先背 VLOOKUP，而是先搞清楚：你拿什麼值去找？去哪一欄找？要回傳哪一欄？找不到時怎麼顯示？

先分 4 種查找情境

先判斷環境與資料方向，再選工具
你的情境優先用一句提醒
Microsoft 365 / Excel 2021 以上XLOOKUP查找欄和回傳欄分開寫，不用數第幾欄
舊版 Excel 或要相容舊檔INDEX + MATCH穩、可往左查，是舊版環境的專業備案
維護既有 VLOOKUP 公式VLOOKUP確認第四參數是 FALSE / 0，且查找欄在表格最左邊
找不到資料時不能出錯XLOOKUP 第 4 參數 / IFERROR不要讓 #N/A 直接出現在要交付的報表裡

語法起手式

# 新版主路線：用產品代碼查產品名稱
=XLOOKUP(A2, 商品表!A:A, 商品表!B:B, "查無此商品")

# 舊版通用備案：MATCH 找位置，INDEX 取結果
=INDEX(商品表!B:B, MATCH(A2, 商品表!A:A, 0))

# 維護舊檔常見：VLOOKUP 只能從最左欄往右查
=VLOOKUP(A2, 商品表!A:E, 2, FALSE)

# 找不到時不要直接露出 #N/A
=IFERROR(VLOOKUP(A2, 商品表!A:E, 2, FALSE), "查無此商品")

3 個最容易搞混的地方

VLOOKUP 的第四參數不要省略

職場查商品、員工、客戶代碼時，幾乎都要精確比對。第四參數請填 FALSE 或 0；省略時會變成模糊比對，資料沒排序時很容易回傳錯人。

VLOOKUP 的欄位號會被插欄弄壞

=VLOOKUP(A2, A:E, 4, FALSE) 的 4 是手動數出來的。中間插入或刪除欄位後，公式可能還能算，但回傳的欄位已經錯了。

INDEX + MATCH 不是舊工具，而是備援思路

MATCH 先找相對位置，INDEX 再依位置取值。這個拆法讓你可以往左查，也更容易理解查找公式到底在做什麼。

實戰範例：同一張商品表的 4 種任務

# A欄=產品代碼，B欄=產品名稱，D欄=單價，E欄=庫存
=XLOOKUP("P003", A:A, B:B, "查無此商品")          # 查名稱
=XLOOKUP("P003", A:A, D:D, "查無此商品")          # 查單價
=INDEX(A:A, MATCH("機械鍵盤", B:B, 0))             # 已知名稱反查代碼
=IFERROR(VLOOKUP("P999", A:E, 2, FALSE), "無此商品") # 舊檔錯誤處理

哪個該用？

M365 / Excel 2021 以上：優先用 XLOOKUP。需要維護舊版環境：熟 INDEX + MATCH。VLOOKUP 不是不能用，而是應該被放在「維護舊檔」和「相容需求」的位置。
