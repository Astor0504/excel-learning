命名範圍與表格

本課進度 0 / 4
TL;DR — 30 秒看懂

命名範圍 + 表格：讓公式變得「會說人話」

=SUMIFS(C2:C500, A2:A500, "電子") vs =SUMIFS(業績[金額], 業績[部門], "電子")——後者半年後回來看也懂。

命名範圍

選範圍 → 左上角「名稱方塊」輸入名字 → Enter。之後公式可以直接寫名字。例如把 B2:B100 取名為「業績」，公式變成 =SUM(業績)。

表格（最強功能之一）

# 選資料 → ⌘+T → 套用表格樣式
# 表格會自動：
# 1. 加標題排序篩選按鈕
# 2. 公式自動展開（新增一列就自動套用）
# 3. 範圍動態擴展（新資料樞紐自動更新）

# 結構化參照（公式裡用欄位名）
=SUM(業績表[金額])
=SUMIFS(業績表[金額], 業績表[部門], "電子")
=業績表[@金額] * 0.05  # @ 代表本列

小技巧

養成把所有資料先轉成「表格」的習慣，後面所有函數和樞紐都會變得超穩定。
