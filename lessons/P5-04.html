綜合挑戰：跨章節整合題目練習

把學過的全部用上！這些題目模擬真實工作場景，每題都會用到 3+ 個函數。

挑戰 1：年資與獎金

員工表有「入職日期」和「年薪」兩欄。要算出：① 滿幾年（DATEDIF）② 年資 ≥ 3 年的人 ③ 每人應發獎金（IFS 分級）④ 全公司獎金加總（SUMIFS）。

挑戰 2：跨表查詢

訂單表只有「商品代號」，商品表有「代號→名稱→價格」。要：① 用 XLOOKUP 帶出商品名 ② 帶出單價 ③ 算小計 ④ 用樞紐做月報。

挑戰 3：動態儀表板

# 給定銷售明細，做一個 Dashboard：
=UNIQUE(銷售[業務員])           # 列出所有業務員
=SUMIFS(銷售[金額], 銷售[業務員], A2)  # 個人業績
=RANK(B2, B:B)                    # 排名
=LARGE(銷售[金額], 1)             # 最高一筆
# 加上條件式格式 + 走勢圖 = 完整儀表板

挑戰 4：自動化清單

每月收到 N 個分公司的 .csv → 用 Power Query 合併 → 用樞紐分析 → 用 VBA 一鍵更新 + 寄信。這是一個完整的「自動化報表系統」。
