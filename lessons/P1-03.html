條件判斷

本課進度 0 / 4
TL;DR — 30 秒看懂

IF 是 Excel 的「if 陳述式」，會了它才能寫條件邏輯

從這課開始 Excel 變成「會思考」的工具。IF 加上 AND/OR 幾乎可以表達所有判斷。

語法

=IF(條件, 條件成立的值, 條件不成立的值)

# 範例：成績及格判斷
=IF(B2>=60, "及格", "不及格")

# 多重條件：用 IFS（比巢狀 IF 好讀）
=IFS(B2>=90, "A", B2>=80, "B", B2>=70, "C", TRUE, "F")

# 複合條件：AND / OR
=IF(AND(B2>=60, C2>=60), "兩科都過", "補考")
=IF(OR(D2="VIP", E2>10000), "享折扣", "原價")

IFS 會「由左到右」檢查條件，一找到 TRUE 就回傳，不繼續看後面。

巢狀 IF vs IFS

舊寫法 =IF(A>90,"A",IF(A>80,"B",IF(A>70,"C","F"))) → 括號地獄。新版直接用 IFS 一行搞定，最後一個 TRUE 當 else。

小技巧

MOD(A2, 2)=0 可以判斷偶數。配合 IF 可以做「奇數列上色」之類的判斷。
